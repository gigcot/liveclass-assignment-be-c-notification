# LAB-001: 알림 도메인 모델 설계

## Success Criteria
- 상태 전이 규칙(PENDING→QUEUED→SENT/FAILED)이 도메인 모델 안에서 강제된다
- 잘못된 상태 전이 시 예외가 발생한다
- 재시도 3회 초과 시 자동으로 FAILED 처리된다
- 읽음 처리가 멱등하다 (중복 호출 시 상태 변경 없음)
- 템플릿 플레이스홀더가 referenceData로 정상 치환된다
- 단위 테스트가 모든 상태 전이 경로와 엣지 케이스를 검증한다
